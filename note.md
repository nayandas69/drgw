1. **Go to GitHub Settings**

1. Visit: [https://github.com/settings/tokens](https://github.com/settings/tokens)
2. Or: GitHub.com → Click your profile picture → Settings → Developer settings → Personal access tokens → Tokens (classic)



2. **Generate New Token**

1. Click "Generate new token" → "Generate new token (classic)"



3. **Configure Token**

1. **Note**: `Discord Role Guardian Website API`
2. **Expiration**: Choose "No expiration" or "90 days" (your choice)
3. **Select scopes** (permissions needed):

1. ✅ `public_repo` (to read public repository data)
2. ✅ `read:user` (to read user profile data)
